# Changelog

## [0.1.0] - 2026-04-22

### Added
- `curate dedup` — hash-based deduplication with multiple strategies
- `curate sort` — auto-detect file type and organize (documents, music, pictures, videos)
- `curate clean` — remove junk files, empty directories, fix permissions
- `curate snapshot` — file system inventory with diff support
- `curate consolidate` — merge source into target with deduplication
- Core library: hashing, transaction logging, progress tracking, safety, permissions
- 109 tests
- Claude Code skill integration
